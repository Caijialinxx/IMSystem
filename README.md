# IMsystem
本系统无需注册，只需提供唯一的昵称即可登录使用。

功能：添加好友、加入/创建群聊、单人/群组聊天、发送文本消息/表情/文件、查询天气。

项目已部署到 [Heroku](https://www.heroku.com/) 上，快来 [参观一下](https://arcane-mountain-17783.herokuapp.com/) 吧！欢迎踊跃指正批评~
